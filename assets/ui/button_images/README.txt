Tool button artwork placeholder directory
=========================================

Each toolbox button expects a 220x120 image stored in this folder. Use the
following filenames when you provide your own art so the launcher will pick
it up automatically:

- ``default.png`` – optional global fallback image
- ``audio_map.png`` – Create Basic Audio Map tool button
- ``hex_glitch.png`` – Hex Glitch tool button
- ``timed_action_mixer.png`` – Timed Action Mixer tool button

If any of these files are missing the launcher will automatically draw a
blank placeholder button that still shows the button text, so you can keep
working without the art files present.
